Portfolio Project

I developed my portfolio website by maximizing the use of graphic tools like Canva and Figma to create a captivating and professional design.

To ensure an optimal user experience across different devices, I implemented a 100% responsive design using HTML and CSS.

Among the techniques employed to make the site adaptable to various screen sizes, I utilized the 'clamp' function to dynamically manage font sizes and media queries to adjust the layout based on device characteristics.
